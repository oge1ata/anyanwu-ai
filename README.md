# Anyanwu AI 

**A Personal AI Coach for Learning, Growth, and Technical Excellence**

## Overview

Anyanwu AI is an intelligent coaching system designed to support individuals navigating technology, learning, and career growth. It acts as a **structured AI coach**, helping users think clearly, learn intentionally, and build consistently across technical and professional goals.

Rather than being a generic chatbot, Anyanwu AI is built to reflect **guided reasoning, accountability, and personalised learning paths**.

## What Problem This Solves

Many learners and early-career technologists struggle with:

* Information overload
* Lack of structure and follow-through
* Unclear learning paths
* Inconsistent motivation

Anyanwu AI addresses this by combining:

* Clear goal setting
* Structured study plans
* Reflective check-ins
* Practical, step-by-step guidance

## Core Capabilities

* **Personalised Learning Paths**

  * Tailored guidance based on goals (AI, software engineering, cloud, etc.)
* **Study & Accountability Coaching**

  * Daily and weekly plans
  * Progress tracking prompts
* **Technical Mentorship Support**

  * Explains concepts clearly
  * Encourages system-level thinking
* **Reflection & Growth**

  * Helps users analyse decisions, habits, and trade-offs

## Design Philosophy

* Coach-first, not tool-first
* Clarity over complexity
* Guidance over answers
* Growth-oriented feedback

This project prioritises **thinking frameworks and decision-making**, not just raw information delivery.

## Tech Stack

* **Python**
* LLM-powered reasoning layer
* Prompt engineering & structured coaching flows
* Modular architecture for future extensions

## Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/<your-username>/anyanwu-ai.git
cd anyanwu-ai
```

### 2. Create and activate virtual environment

```bash
python -m venv venv
source venv/bin/activate   # macOS/Linux
# venv\Scripts\activate    # Windows
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```
